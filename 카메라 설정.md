# 카메라 설정



리눅스 시스템에서 작동하는 것이므로 반드시 리눅스에서 실행 가능한 카메라인지를 확인하여야 한다. 일부 중국산 카메라의 경우 리눅스를 지원한다고 하지만 wsl 가상환경에서는 지원이 안되는 경우가 있으므로 주의한다.

## 카메라 패키지 설치

카메라 패키지는 v412_camera 또는 usb_cam패키지 가 있으나 여기서는 가장 안정적인 usb_cam패키지를 실행할 것이다.

상태 업데이트 및 usb_cam패키지 설치#

```
sudo apt update
```

```
sudo apt install ros-humble-usb-cam
```



비디오 확인 (비디오가 여러개일수있으므로 하나씩 확인해봐야한다)

```
ls /dev/video*
```



노드실행 (확인된 비디오의 번호를 맞춰야함)

```
ros2 run usb_cam usb_cam_node_exe --ros-args -p video_device:="/dev/video0"
```

```
sudo apt update
```

시각화 도구 설치 

```
sudo apt install ros-humble-rqt-image-view ros-humble-rqt-common-plugins
```

시각화 도구를 실행하여 확인해본다. 

```
ros2 run rqt_image_view rqt_image_view
```
